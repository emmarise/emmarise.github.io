# emmarise.github.io

Personal website for Furong Tian.

## Development

This is a static site built with pure HTML and CSS. No build step required.

To view locally:
1. Open `index.html` in your browser
2. Or use a local server: `python3 -m http.server 8000`

## Deployment

Automatically deployed to https://emmarise.github.io via GitHub Pages.

To update:
```bash
git add .
git commit -m "Update website"
git push
```

Changes will be live in 1-2 minutes.
