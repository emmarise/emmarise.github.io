A single-page interactive itinerary for a Kauai + Oahu trip (Apr 23–30, 2026). Built entirely through vibe coding with Claude Code in one session. Features geo-projected SVG island maps with hover-able stop markers, color-coded daily timelines, a must-reserve dining tracker, and a logistics checklist — all in one self-contained HTML file with zero dependencies.
